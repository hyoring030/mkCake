# mkCake
#나에게 생일편지를 써줄랭?

-생일과 이름 정보를 데이터베이스(sqlite3)에 저장 후 선택된 초 이미지와 생일 축하 메시지를 남길 수 있는 기능을 제공
-입력된 생일 축하 메시지를 사용자가 읽을 수 있는 웹 사이트를 React를 활용하여 구현
